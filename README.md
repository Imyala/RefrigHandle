RefrigHandle

A mobile and desktop-friendly refrigerant management and compliance system designed for HVAC/R technicians and businesses. RefrigHandle provides accurate refrigerant tracking, usage logging, and audit records to help maintain compliance and improve inventory control.

Built as an installable Progressive Web App (PWA), RefrigHandle works on iPhone, iPad, Android devices, tablets, and desktop computers from a single codebase.

Features
Refrigerant Cylinder Management
Track cylinder serial numbers and IDs
Record refrigerant type (R410A, R32, R134a, R404A, R407C, R22, R290, R600A, R1234yf, R454B, R513A, and more)
Monitor tare weight and current refrigerant quantity
Track cylinder status:
In Stock
On Site
Returned
Empty
Quarantined
Site & Customer Records
Store customer and site information
Track which cylinders are assigned to each location
Maintain service history for every site
Refrigerant Usage Logging

Record all refrigerant movements, including:

Charging equipment
Recovering refrigerant from systems
Cylinder-to-cylinder transfers
Returning cylinders to stock
Manual stock adjustments

Each transaction stores:

Date and time
Technician details
Before and after cylinder weights
Refrigerant quantities moved
Equipment or site reference
Notes and comments
Audit & Compliance Records
Complete transaction history for every cylinder
Full refrigerant movement traceability
Historical records for environmental and regulatory audits
Quick access to usage reports and inventory history
Dashboard & Reporting
Total refrigerant inventory by refrigerant type
Current stock levels across all cylinders
Cylinder status overview
Refrigerant usage summaries
Backup & Data Export
Export complete database backups to JSON
Export transaction records to CSV for reporting
Restore data from previous backups
Offline First

All data is stored locally on the device, allowing RefrigHandle to operate fully offline without requiring an internet connection.
